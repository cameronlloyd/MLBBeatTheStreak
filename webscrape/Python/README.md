1. Make sure python and pip are installed:
	$ apt-get install python
	$ apt-get install python-pip

2. Install the following libraries for python BeautifulSoup webscraping (https://www.crummy.com/software/BeautifulSoup/bs4/doc/#)
	$ apt-get install python-bs4
	$ pip install beautifulsoup4
	$ pip install beautifulscraper
	$ pip install lxml
	$ pip install selenium

3. Within Config.py, modify start and end date for scraper to run 

4. Execute the script by running:
	$ python WebScrape.py

5. Results will be stored in:
	 ./Results/HeadToHead
